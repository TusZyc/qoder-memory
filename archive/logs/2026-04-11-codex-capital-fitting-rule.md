# 2026-04-11 Codex 旗舰级装备规则记录

## 背景
- 在加入“适合当前舰船”和“资源足够”筛选后，继续检查装配规则是否贴近游戏。
- 本次重点是旗舰级、超大型装备，避免普通舰船看到或装上旗舰装备。

## 发现的问题
- 一些旗舰级装备和超大型武器，在 SDE 里没有明确写出只能装在哪些舰船上。
- 典型例子包括：
  - 旗舰级远程装甲维修器。
  - 旗舰级远程护盾、远程电容装备。
  - 超大型炮台分支。
- 因为用户已经明确：未勾选“资源足够”时允许超资源装配，所以不能只靠 CPU 或能量栅格消耗来判断是否合法。

## 已实现规则
- 舰船详情新增：
  - `resources.is_capital_size`
- 该字段来自 SDE 属性 `1785`，表示舰船是否属于旗舰体量。
- 如果装备看起来属于旗舰级或超大型装备，普通舰船在“适合当前舰船”筛选下不会看到它。
- 前端搜索过滤和点击安装也使用同一条规则。
- 硬点规则保持不变：
  - 硬点不参与列表筛选。
  - 只有用户真正安装炮台或发射器时，才提示并阻止硬点不足的情况。

## 服务器验证
- 裂谷级 / `587`：
  - `resources.is_capital_size = false`
  - 超大型火炮叶子分类返回 `0`
  - 旗舰级远程装甲维修叶子分类返回 `0`
- 长须鲸级 / `28352`：
  - `resources.is_capital_size = true`
  - 仍能看到旗舰级和超大型装备。

## 影响
- 修复了普通舰船可能看到旗舰装备的问题。
- 这条规则是“舰船体量兼容”规则，不是资源规则，也不是硬点规则。
