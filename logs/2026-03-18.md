#### 证书申请
```bash
# 配置阿里云 DNS API 密钥
export Ali_Key='你的AccessKeyId'
export Ali_Secret='你的AccessKeySecret'
# 申请证书
~/.acme.sh/acme.sh --issue --dns dns_ali -d 51-eve.online -d '*.51-eve.online' --keylength 2048
```

#### 证书安装
```bash
~/.acme.sh/acme.sh --install-cert -d 51-eve.online \
  --key-file /etc/nginx/ssl/51-eve.online.key \
  --fullchain-file /etc/nginx/ssl/51-eve.online.crt \
  --reloadcmd 'docker restart eve-esi-nginx'
```

## 遇到的问题

### Nginx 配置变量展开问题
- **问题**: SSH 传输 Nginx 配置时，`$host`、`$request_uri` 等变量被本地 shell 展开
- **解决**: 使用 base64 编码传输配置文件

### Nginx 版本兼容问题
- **问题**: Nginx 1.29.6 不支持 `Cache-Control` 的 `immutable` 参数
- **解决**: 移除 `immutable` 参数
